The `siteurl` shortcode is used to help you print the current base URL for your hosted site. 

For instance, in this current site, the baseURL is {{% siteurl %}}

Use the `siteurl` shortcode to display its value.

```
{{%/* siteurl */%}}
```

is displayed as

{{% siteurl %}}
